## Usage

```bash
# npm
npm init @netless/app [--template vanilla]
# pnpm
pnpm init @netless/app [--template vanilla]
# yarn
yarn create @netless/app [--template vanilla]
```

Currently supported template presets include:

- `vanilla`
- `svelte`
